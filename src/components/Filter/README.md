# Filter 컴포넌트

## 역할
복지 서비스 필터링 UI. 나이대, 상황, 지역을 조합하여 맞춤 복지를 필터링한다.

## Props
| Props | 타입 | 설명 |
|---|---|---|
| onFilterChange | (filter: FilterType) => void | 필터 변경 이벤트 |
| defaultFilter | FilterType | 기본 필터 값 |

## 주요 기능
- 나이대 필터 (영유아 / 청년 / 장년 / 노년)
- 상황 필터 (신혼 / 임산부 / 취업 / 장애 / 저소득)
- 지역 필터 (시/구 단위)
- 필터 초기화 버튼

## 접근성 고려사항
- 버튼형 필터 (탭 키 접근 가능)
- 선택된 필터 명확한 시각적 표시
- 큰 터치 영역 확보
