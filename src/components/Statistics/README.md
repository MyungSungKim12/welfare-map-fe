# Statistics 컴포넌트

## 역할
현재 선택된 지역의 복지 서비스 통계를 시각적으로 보여주는 섹션.
숫자 카드 + 카테고리별 분포로 구성한다.

## 표시 데이터
| 항목 | 설명 |
|---|---|
| 전체 서비스 수 | 현재 지역 총 복지 서비스 건수 |
| 카테고리별 수 | 의료 / 주거 / 교육 / 취업 / 보건 |
| 이번 달 신규 | 이번 달 새로 등록된 서비스 수 |
| 마감 임박 | 30일 이내 마감 예정 서비스 수 |

## 주요 기능
- 숫자 카드 4개 가로 배치
- 카테고리별 서비스 수 바 형태 표시
- 지역 변경 시 통계 자동 업데이트

## 디자인
- 배경: #F7F4EF (크림톤)
- 숫자 카드: 흰 배경, 포인트 컬러 숫자
- 포인트 컬러: #2E9E7A / #1B3A4B
