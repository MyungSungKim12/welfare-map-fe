# PopularList 컴포넌트

## 역할
인기/추천 복지 서비스 섹션.
많이 조회된 복지 TOP5와 마감 임박 복지를 가로 스크롤 카드로 표시한다.

## 탭 구성
| 탭 | 내용 |
|---|---|
| 많이 본 복지 | 이번 달 조회수 TOP 5 |
| 마감 임박 | 신청 마감일 기준 오름차순 |
| 신규 등록 | 최근 등록된 복지 |

## 주요 기능
- 탭 전환 방식
- 가로 스크롤 카드 (overflow-x: auto)
- 마감 임박 빨간 배지, 신규 초록 배지 표시

## 디자인
- 배경: #ffffff
- 카드: 콤팩트한 사이즈 (세로 스크롤 방지)
- 배지: 마감임박 #E74C3C / 신규 #2E9E7A

## 접근성 고려사항
- 가로 스크롤 시 키보드 방향키 지원
- 카드 내 핵심 정보만 노출 (간결하게)
