# LifeStage 컴포넌트

## 역할
생애주기별 빠른 진입 섹션.
아이콘 카드를 클릭하면 해당 필터가 자동 적용된 복지 리스트로 이동한다.

## 생애주기 구분
| 유형 | 아이콘 | 필터 적용 |
|---|---|---|
| 영유아 | 👶 | ageGroup: child |
| 청년 | 🎓 | ageGroup: youth |
| 신혼부부 | 💑 | situation: newlywed |
| 중장년 | 👔 | ageGroup: middle |
| 노년 | 🧓 | ageGroup: senior |
| 장애인 | ♿ | situation: disability |

## 주요 기능
- 아이콘 카드 6개 가로 배치 (grid)
- 클릭 시 해당 필터 적용 후 리스트 섹션으로 스크롤
- 호버 시 카드 강조 효과

## 디자인
- 배경: #F7F4EF (크림톤)
- 카드: 흰 배경, 라운드 코너
- 포인트: #2E9E7A

## 접근성 고려사항
- 아이콘 + 텍스트 병행 표시 (아이콘만 단독 사용 금지)
- 카드 최소 높이 100px (터치 영역 확보)
- 글자 크기 최소 16px
