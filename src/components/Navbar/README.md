# Navbar 컴포넌트

## 역할
페이지 상단 네비게이션 바. 로고, 현재 위치 표시, 로그인 버튼을 포함한다.

## Props
| Props | 타입 | 설명 |
|---|---|---|
| location | string | 현재 사용자 위치 (시/구 단위) |
| isLoggedIn | boolean | 로그인 여부 |

## 주요 기능
- 로고 클릭 시 메인 페이지 이동
- 현재 위치(GPS) 표시
- 로그인 / 마이페이지 버튼
- 반응형 (모바일 햄버거 메뉴)

## 접근성 고려사항
- 글자 크기 최소 16px 이상
- 버튼 클릭 영역 최소 48px 이상
- 고대비 색상 사용
