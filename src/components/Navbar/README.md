# Navbar 컴포넌트

## 역할
페이지 상단 네비게이션 바. 로고, 현재 위치 선택, 로그인 버튼을 포함한다.
위치는 GPS 자동 감지 또는 사용자가 직접 시/구 단위로 선택할 수 있다.

## Props
| Props | 타입 | 설명 |
|---|---|---|
| isLoggedIn | boolean | 로그인 여부 |

## 주요 기능
- 로고 클릭 시 메인 페이지 이동
- 현재 위치 표시 버튼 클릭 시 위치 변경 모달 오픈
- GPS 자동 감지 / 직접 선택 (시/도 → 시/군/구)
- 로그인 / 로그아웃 / 마이페이지 버튼
- 미로그인 시 로그인 버튼, 로그인 시 마이페이지 아이콘

## 디자인
- 배경: #1B3A4B (딥 네이비)
- 텍스트: #ffffff
- 위치 배지: rgba(255,255,255,0.12) 반투명

## 접근성 고려사항
- 글자 크기 최소 16px 이상
- 버튼 클릭 영역 최소 48px 이상
- 고대비 색상 사용
