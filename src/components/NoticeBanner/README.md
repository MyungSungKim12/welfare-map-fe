# NoticeBanner 컴포넌트

## 역할
공지사항 및 안내 배너 섹션.
신규 복지 안내, 마감 임박 알림, 복지 신청 가이드를 제공한다.

## 배너 유형
| 유형 | 내용 |
|---|---|
| 신규 | 이번 달 새로 등록된 복지 서비스 안내 |
| 임박 | 신청 마감이 가까운 복지 알림 |
| 가이드 | 복지 신청 방법 간단 안내 (고령자 배려) |

## 주요 기능
- 배너 슬라이드 또는 카드 형태 표시
- 클릭 시 해당 복지 상세 페이지 이동
- 신규/임박/가이드 뱃지 구분 표시

## 접근성 고려사항
- 자동 슬라이드 시 정지 버튼 제공
- 충분한 글자 크기 (16px 이상)
- 색상 외 텍스트로도 유형 구분
