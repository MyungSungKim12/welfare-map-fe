# Footer 컴포넌트

## 역할
페이지 하단 푸터. 서비스 소개, 관련 기관 링크, 문의처를 포함한다.

## 구성 요소
| 영역 | 내용 |
|---|---|
| 서비스 소개 | 복지맵 소개 문구, 로고 |
| 관련 기관 | 복지로, 정부24, 공공데이터포털 링크 |
| 법적 고지 | 개인정보처리방침, 이용약관 |
| 문의 | 이메일 주소 |

## 주요 기능
- 관련 기관 외부 링크 (새 탭으로 열기)
- 개인정보처리방침 / 이용약관 링크
- Copyright 표시

## 디자인
- 배경: #1B3A4B (딥 네이비)
- 텍스트: rgba(255,255,255,0.7)
- 링크 hover: #ffffff
