# Hero 컴포넌트

## 역할
메인 페이지 최상단 히어로 섹션.
서비스 소개 문구, 검색바, CTA 버튼, 서비스 통계 수치를 포함한다.

## 주요 기능
- 메인 타이틀 / 서브 타이틀 표시
- 복지 서비스명 검색바
- "지금 내 위치로 복지 찾기" CTA 버튼 → GPS 위치 자동 감지 후 필터 적용
- 서비스 통계 수치 표시 (전국 복지 서비스 수 등)

## 디자인
- 배경: #1B3A4B (딥 네이비) 풀너비
- 텍스트: #ffffff
- CTA 버튼: #2E9E7A (에메랄드 그린)
- 최대 너비: 1280px 중앙 정렬

## 접근성 고려사항
- 타이틀 최소 32px 이상
- 검색바 높이 최소 52px (고령자 터치 편의)
- 배경 대비 텍스트 가독성 확보
