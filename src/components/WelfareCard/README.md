# WelfareCard 컴포넌트

## 역할
복지 서비스 한 건의 정보를 카드 형태로 표시하는 컴포넌트.

## Props
| Props | 타입 | 설명 |
|---|---|---|
| welfare | WelfareItem | 복지 서비스 데이터 |
| onSave | (id: string) => void | 저장 버튼 클릭 이벤트 |
| isSaved | boolean | 저장 여부 |

## 주요 기능
- 복지 서비스 요약 정보 표시
- 대상 / 지역 배지 표시
- 저장 버튼 (로그인 시 활성화)
- 상세보기 링크

## 접근성 고려사항
- 글자 크기 18px 이상
- 카드 전체 클릭 가능
- 색상 외에도 텍스트로 상태 표시
